# 📊 AI Resume Screening & Skill Analyzer

An AI-powered Resume Analyzer web application that analyzes resumes based on ATS standards, job-role matching, resume strength, skill gaps, and AI suggestions.

---

# 🚀 Features

- Upload Resume PDF
- ATS Resume Analysis
- Resume Strength Score
- Role Match Prediction
- Missing Skills Detection
- AI Suggestions
- Suggested Courses
- HR Report Generation
